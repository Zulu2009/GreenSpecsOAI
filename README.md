# GreenSpecsOAI

## YKO Local Intelligence Staging (SQLite)

This repository now includes a **local-only** SQLite staging workflow for YKO intelligence data.

- ✅ Local database path: `yko_local_intel/data/yko_staging.sqlite`
- ✅ Separate from production systems
- ✅ No live endpoint modifications
- ✅ No production connections in the tooling

## Folder Structure

```text
.
├── Makefile
├── README.md
└── yko_local_intel
    ├── data
    │   ├── brands.csv
    │   └── yko_staging.sqlite          # generated locally after init
    ├── exports
    ├── json_snapshots
    │   └── yko_snapshot.json           # generated by export-json
    ├── scripts
    │   └── yko_staging_db.py
    └── sql_exports
        └── yko_inserts.sql             # generated by export-sql
```

## Commands

### 1) Initialize local DB

```bash
make yko-init
```

### 2) Import brands from CSV

```bash
make yko-import-brands
```

### 3) Add claims and evidence later

```bash
make yko-add-claim BRAND="Clover Sonoma" TEXT="Uses rBST-free milk" STATUS="pending" SOURCE="internal_research"

make yko-add-evidence CLAIM_ID=1 TEXT="Company sustainability report 2025" TYPE="document" SOURCE_URL="https://example.org/report.pdf"
```

### 4) Export JSON snapshots

```bash
make yko-export-json
```

### 5) Export SQL INSERTs for Cloudflare D1 import

```bash
make yko-export-sql
```

## Direct script usage

You can also use the script directly:

```bash
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite init-db
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite import-brands --csv yko_local_intel/data/brands.csv
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite add-claim --brand "Amy's Kitchen" --text "No artificial preservatives" --status pending
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite add-evidence --claim-id 1 --text "Label scan confirms claim" --type label
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite export-json --out yko_local_intel/json_snapshots/yko_snapshot.json
python3 yko_local_intel/scripts/yko_staging_db.py --db yko_local_intel/data/yko_staging.sqlite export-sql --out yko_local_intel/sql_exports/yko_inserts.sql
```
